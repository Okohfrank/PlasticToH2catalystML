# PlasticToH₂ CatalystML

> Predicting photocatalytic hydrogen production from plastic waste 
> using Machine Learning — R² = 0.9827

---

## Problem Statement

Plastic waste pollution and clean energy scarcity are two of the 
defining challenges of our era. Photocatalytic hydrogen production 
from plastic waste addresses both simultaneously — but catalyst 
discovery through conventional experimental methods is slow, 
expensive, and resource-intensive.

This project applies machine learning to accelerate that process, 
building a predictive model that estimates hydrogen yield based on 
catalyst and process parameters — replacing trial-and-error with 
data-driven decisions.

---

## Results

| Metric | Value |
|---|---|
| Model | Random Forest Regressor |
| R² Score | **0.9827** |
| Accuracy | **98.3%** |
| Dataset | Synthetic, literature-informed |

---

## Methodology

1. **Dataset Construction** — Built a synthetic dataset informed 
   by published literature on photocatalytic H₂ production systems
2. **Feature Engineering** — Selected catalyst properties and 
   process parameters as input features
3. **Model Development** — Trained and evaluated a Random Forest 
   Regressor using scikit-learn
4. **Validation** — Assessed model performance using R² score 
   and cross-validation techniques
5. **Visualization** — Generated prediction plots and feature 
   importance analysis using matplotlib
